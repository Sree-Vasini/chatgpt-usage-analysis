# ChatGPT Usage Analysis

## Project Overview
This project analyzes my personal ChatGPT usage using Python.  
I tracked and categorized my last 10 ChatGPT conversations to understand how AI helps me in daily tasks.

The goal was to:
- Measure how much time ChatGPT saves me
- Understand which categories I use ChatGPT for the most
- Visualize my AI usage patterns

---

## Dataset

The dataset was manually created based on my last 10 conversations.

### Columns in the dataset:

- Category: Type of task (Learning, Career, Writing, Personal, Coding)
- Time Saved: Estimated minutes saved using ChatGPT
- Used Output: Whether I used the generated response
- Notes: Short description of the conversation

File location: `data/chatgptdata.csv`
